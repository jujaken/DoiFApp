
привет, это приложение было создано для К.
используется MVVM архитектура, для упрощения использования - CommunityToolkit.Mvvm

поскольку требовалось большое число как бы конвертеров типов данных (как между самими типами, так и между форматами), были изобретены Services/Data интерфейсы
Sqlite используется для хранения данных в сессии приложения. это позволяет пользователю по сто раз не загружать одно и то же, doifapp.db

страницы подвязываются через PagesTamplate.xaml
